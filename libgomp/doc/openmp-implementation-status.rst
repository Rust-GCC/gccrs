OpenMP Implementation Status
----------------------------

The ``_OPENMP`` preprocessor macro and Fortran's ``openmp_version``
parameter, provided by ``omp_lib.h`` and the ``omp_lib`` module, have
the value ``201511`` (i.e. OpenMP 4.5).
