.. _acc_hostptr:

acc_hostptr -- Get host pointer associated with specific device address.
************************************************************************

Description
  This function returns the host address that has been mapped to the
  device address specified by :samp:`{d}`.

C/C++:
  .. list-table::

     * - *Prototype*:
       - ``void *acc_hostptr(d_void *d);``

Reference:
  :openacc:`2.6`, section
  3.2.29.
