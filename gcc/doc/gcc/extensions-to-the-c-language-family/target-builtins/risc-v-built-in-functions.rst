RISC-V Built-in Functions
^^^^^^^^^^^^^^^^^^^^^^^^^

These built-in functions are available for the RISC-V family of
processors.

.. function:: void * __builtin_thread_pointer (void)

  Returns the value that is currently set in the :samp:`tp` register.
